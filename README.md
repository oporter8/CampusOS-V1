# CampusOS v1

An AI-powered operating system for students.

## Features

- AI Tutor
- Research Assistant
- Assignment Planner
- Internship Finder
- Knowledge Graph
- Multi-Agent System

## Architecture

Frontend:
- Next.js
- TypeScript
- Tailwind

Backend:
- NestJS
- PostgreSQL
- Redis

AI:
- OpenAI
- LangGraph
- Vector Database

## Quick Start

git clone ...
pnpm install
docker compose up -d
pnpm dev
